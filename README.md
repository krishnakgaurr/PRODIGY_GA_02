# PRODIGY_GA_03 - Text Generation with Markov Chains

## Objective

Develop a text generation application using Markov Chains that learns word transition probabilities from a text corpus and generates new text.

## Technologies Used

- Python 3.12
- Markovify

## Project Structure

```
PRODIGY_GA_03/
│
├── dataset/
│   └── corpus.txt
│
├── outputs/
│   └── generated_text.txt
│
├── main.py
├── requirements.txt
├── README.md
└── .gitignore
```

## Installation

```bash
git clone <repository-url>
cd PRODIGY_GA_03

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python main.py
```
